这是一个专门为 **《恋与制作人》** 做的日常脚本

---

### ✨ 它有什么特别的？

* **它有一双“机器眼”**：不同于死板的坐标点击，它会盯着屏幕看！只要认出了我教给它的图标（比如公司按钮、收菜小红点），它就会精准地“哒”一下。
* **不怕点歪**：内置了“9点特征识别”和“坐标等比缩放”，就算模拟器窗口拉大拉小，它也能自己算清楚该点哪。
* **会自救**：如果某个任务卡住了，它等几秒没反应会自己尝试滑动屏幕或者跳过，主打一个“省心”。
* **防屏蔽**：模拟了真人的按压时长（约 150ms），完美绕过游戏引擎的检测。

---

### 🚀 怎么使用？

1. **环境准备**：建议使用 **MuMu 模拟器**，分辨率设置为 **1920x1080**。
2. **下载安装**：在右侧 [Releases](https://github.com/Aluovo/autoclicker/releases) 下载最新的 APK 安装到模拟器里。
3. **开启权限**：启动后给它“无障碍服务”和“录屏”权限（它是靠录屏来“看”画面的）。
4. **开始挂机**：点击悬浮窗的“134步序列找图”，然后切回游戏主界面，剩下的交给它就好啦！
